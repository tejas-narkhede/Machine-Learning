
# Crop Recommendation System  

**Hello Everyone**,  
During my 4<sup>th</sup> semester, I had an incredible learning experience while collaborating with three teammates on a project titled **Crop Recommendation System** as part of Project-Based Learning.  

## Project Overview  
As second-year students at the time, we were beginners in many technologies. However, we eagerly explored different tech stacks and learned how to integrate them to create a fully functional website.  

This project was designed with farmers in mind. By inputting attributes like **Nitrogen, Phosphorus, Potassium, humidity, temperature, and soil moisture**, farmers could predict the most suitable crop for their harvest. We developed a user-friendly website using:  
- **Frontend**: HTML, CSS  
- **Backend**: Flask  

## My Contributions  
I was responsible for implementing a suitable **Machine Learning (ML) algorithm**. Being a beginner, I researched various options such as:  
- Logistic Regression  
- K-Means Clustering  
- Support Vector Machines (SVM)  
- Ensemble methods like Decision Trees and Random Forest  

After evaluating the accuracy and feasibility of these algorithms, I chose **Random Forest**, as it provided the best results. This was my first ML algorithm implementation, and it was a significant milestone for me.  

## Learning Journey  
Initially, I faced challenges due to my limited knowledge of ML concepts and ensemble methods. However, I overcame these hurdles by:  
- Referring to online documentation for **NumPy**, **Pandas**, and **Scikit-learn**  
- Learning foundational ML concepts from Josh Starmer's book, **StatQuest**  
- Experimenting through trial and error  
- Seeking assistance from ChatGPT  

This experience not only enhanced my technical skills but also taught me the value of perseverance and collaboration.
